Explications sur le tar.gz
.tar est un type de fichier comprenant une liste de fichiers non compressés. 
.gz est un format compressé.

Commande linux pour extraire un tar.gz : tar xzf nomDuFichier

En réalité, les formats visibles sur Windows ne sont pas forcémment le réel type de fichier en question.

Quelques infos en réseau :

Socket = IP + port
Port = service / processus (route pour atteindre le logiciel qui tourne)

Le serveur DHCP fournit les adresses IP aux machines sur le réseau avec une table sous forme : MAC / IP

Le FTP est un protocole vieux qui ne sécurise pas les données transmises et nécessite l'utilisation de deux ports (20 et 21)
Le protocole SSH (SCP et SFTP) est à privilégier car il utilise un seul port (22) et sécurise les données envoyées et reçues.

Commandes Linux :

-history : historique de commandes
-vi / vim : éditeur de texte
-rm : supprime un fichier
-grep : filtre avec le nom donné en argument

| = entrée / sortie pour plusieurs commandes
