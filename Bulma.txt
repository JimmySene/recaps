Bulma est un framework CSS plus léger que Bootstrap et ne faisant pas appel à Javascript. Ce qui améliore les performances.
Le framework est constitué d'un seul fichier CSS possédant de nombreuses class à utiliser sur nos éléments HTML.

Importer via CDN : <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/bulma/0.7.1/css/bulma.min.css">

La balise meta responsive est nécessaire au bon fonctionnement du framework : <meta name="viewport" content="width=device-width, initial-scale=1">
