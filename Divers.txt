Multiplexage :
	-Faire passer plusieurs informations à travers un seul support de transmission.
Shadow DOM : 
	-Permet d'encapsuler du JavaScript et du CSS au sein d'un Web Component
	-Séparation du DOM principal
	-Utilisable seul en dehors du component
	Utiliser la méthode .attachShadow({mode: "open"}) sur un élément pour appliquer le SHADOW DOM dessus.
	Accessible via le noeud racine ShadowRoot.




Serveur : Ordinateur relié à Internet ayant pour rôle de servir des données à celui qui lui en demande.




Sérialisation : Consiste à transformer une structure de donnée simple ou complexe en fichier texte.

Formats de sérialisation : 

-Format CSV :
	"Comma Separated Value". Les données sont séparées par des virgules.
	Fichiers principalement générés par Excel et Calc (Tableurs).
-Format YAML : (Yet Another Markup Language)
	standard de sérialisation de données pour tous les langages, facile à comprendre pour les humains
	Permet d'inclure plusieurs documents en les séparant par ---
	S'utilise davantage pour la configuration de fichiers.
	Continet des types avancés comme les dates
	Commentaires : #
	Exemple :
	---
	nom_section:
		nom: valeur
		nom: valeur

	autre_section:
		nom: valeur
		...
-Format XML : 
	reine des formats de sérialisation. Pro mais verbeux, fragile et complexe.
	Données entre balises <>.
-Format JSON (Javascript Object Notation) :
	Représentation des données sous forme de variables javascript. Plus légère que XML. Pas de commentaires.
	Subset de YAML.
	S'utilise davantage pour l'envoi de données.
	Se présente sous cette forme :
	[
		{
			'nom' : 'valeur',
			'nom' : 'valeur'
		},
		{
			...
		}
	]
-Format INI
	Souvent des fichiers de configuration avec des paramètres
	Constitué de deux éléments : l'en-tête (ou section) s'écrivant [nom_section] ; et la valeur s'écrivant nom=valeur
	Exemple :
	[nom_section]
	nom=valeur
	nom=valeur

	[autre_section]
	nom=valeur
	...

Dart (anciennement appelé "Dash") : Langage de programmation créé par Google en 2011, visant à remplacer Javascript. 
Cependant, le but initial semble avoir été abandonné et la priorité est que le code dart soit compilé en JS sur tous les navigateurs.

Typescript : Langage de programmation créé par Microsoft en 2012, visant à améliorer et sécuriser le développement en langage Javascript.
Il permet le typage statique, la création de classes et d'interfaces et l'implantation de modules en gardant l'approche non contraignante du JS.

EcmaScript : Standard du web et ensemble de normes imposées aux langages de script. Langage de programmation orienté prototype.

Programmation orienté prototype : Programmation orienté objet sans classes. Un prototype est un objet à partir duquel on peut créer d'autres objets.

Langage de haut niveau : Langage avec un haut niveau d'abstraction et compréhensible pour un humain.

Langage de bas niveau : Langage avec un bas niveau d'abstraction et difficilement compréhensible pour un humain, qui se rapproche plus du langage machine.

Assembleur : Langage de bas niveau représentant le langage machine lisible pour un humain.

Compilateur : Programme qui transforme un code source en un langage de bas niveau pouvant être lu par la machine.

Décompilateur : Programme qui transforme du langage de bas niveau en langage de haut niveau.

Transcompilateur (compilateur source à source) : Programme qui traduit un code source vers un autre langage de programmation (haut niveau).

Origine : Combinaison d'un protocole, un hôte et un numéro de port
